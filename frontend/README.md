# PUPQuestC — Frontend (React App)

> Part of the **ADET Project** | BSIT — Polytechnic University of the Philippines QC  
> **Developer:** Reca Maelah Conag

This is the React + TypeScript frontend for the PUPQuestC Lost & Found platform.

---

## How to Run

```bash
npm install
npm run dev
```

App runs at **http://localhost:5173**

Make sure the backend server is also running at `http://localhost:5000` before starting the frontend.

---

## Tech Stack

React 18, TypeScript, Tailwind CSS, Flowbite React, Redux Toolkit (RTK Query), React Hook Form + Zod, Framer Motion
